日本各个大学英语项目总结与信息互享
本项目旨在信息共享，拒绝被中介骗，同时也不影响申请者的公平公正。
欢迎各位积极参与该项目建设，如果有不会用github的同学，请加qq群580719467私戳github项目管理员。在上传申请情况时禁止提及具体考核内容（题目的具体情况或者具体知识点），可以共享考试范围（如线代，微积分等）。如有发现违规，管理者会发邮件给该教授检举违规行为。

文件组织方式：

	各个大学英语项目官网.txt
	大学名称/
		
	
		院系名称/
	
			申请情况.txt
